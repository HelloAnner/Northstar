# 数据校准与目标对齐辅助工具 - 业务背景与口径

## 1. 项目背景（来自 PRD + PDF）
- 业务痛点：每月 240 家左右企业、4 个核心表格需要人工汇总、反复试算、调整后再验证，耗时 3-4 小时，错误率高。
- 目标：把“手工试错调数”变成“自动联动+快速调整”，将工作耗时压缩到 20 分钟内，同时保持结果可控。
- 工具定位：离线桌面工具，读取 Excel 多 sheet，实时计算、联动调整、导出规范格式，最终输出与客户日常“定稿版”完全一致。

## 2. 业务对象与核心概念
- **限上企业**：规模较大的重点企业，是主要操作对象。
- **限下企业**：小微企业抽样估算，增速由“限上小微企业”推导。
- **小微企业**：单位规模字段为 3/4。
- **吃穿用企业**：按行业代码与商品分类规则识别。
- **四大行业**：批发、零售、住宿、餐饮。
- **当月值/累计值**：当月值为当前月份数据，累计值为 1-当月的累计。

## 3. 指标口径（16 项基础指标）
> 这些指标是计算引擎的核心联动结果，任何底层数据变化都需要实时更新。

### 3.1 限上社零额（4项）
1) 限上社零额(当月) = SUM(所有企业 本期零售额)
2) 限上社零额增速(当月) = (SUM 本期零售额 - SUM 上年同期零售额) / SUM 上年同期零售额
3) 限上社零额(累计) = SUM(所有企业 本年累计零售额)
4) 限上社零额增速(累计) = (SUM 本年累计零售额 - SUM 上年累计零售额) / SUM 上年累计零售额

### 3.2 专项增速（2项）
5) 吃穿用增速(当月)：吃穿用企业口径（行业/商品分类）
6) 小微企业增速(当月)：单位规模=3/4

### 3.3 四大行业增速（8项）
按批发/零售/住宿/餐饮分别计算当月/累计销售额（或营业额）增速：
- 当月增速 = (SUM 本年当月 - SUM 上年同期) / SUM 上年同期
- 累计增速 = (SUM 本年累计 - SUM 上年累计) / SUM 上年累计

### 3.4 社零总额（2项）
15) 社零总额(累计)：
- 估算本年累计限下社零额 = 上年累计限下社零额 × (1 + 小微企业增速)
- 社零总额(累计) = 限上社零额(累计) + 估算本年累计限下社零额

16) 社零总额增速(累计)：
- 上年社零总额(累计) = SUM 上年累计零售额 + 上年累计限下社零额
- 累计增速 = (本年社零总额(累计) - 上年社零总额(累计)) / 上年社零总额(累计)

## 4. 二期复杂业务规则（必须进入算法约束）
来自 PDF 末尾规则（对调整算法、联动校验有强约束）：
1) 四大行业：当月同比/环比增速正负均在 30% 内且不同企业增速不能雷同（控制在 1% 浮动区间）。
2) 批发业：零销比一般不超过 40%，大个体不超过 30%，且不能雷同（小数点后两位不相同）。
3) 零售业：乡镇加油站零销比一般 50% 左右；其他企业销售额≈零售额且不雷同；大个体增速一般不超过 20%。
4) 住宿业：有餐饮的企业客房收入高于餐费收入。
5) 餐饮业：有住宿的企业客房收入低于餐费收入；纯住餐企业营业额=客房收入或餐费收入。
6) 住宿/餐饮业：销售额与零售额增速要求上下浮动个位数保持不变，小数点后一位变化≤1%。
7) 大个体：增速不能超过法人增速；乡镇超市当月≤100万；当月增速≤20%。
8) 批发业销售额、零售额增速上下浮动个位数保持不变，小数点后一位变化≤1%。
9) 新进企业同期数：12个月累计批发≤2000万、零售≤500万、住餐≤200万。
10) 小微与吃穿用：优先最大化，靠近 30%。

## 5. 输出目标
- 输入：`12月月报（预估）.xlsx`
- 输出：`12月月报（定）.xlsx`
- 要求：sheet 名称、顺序、格式、合并单元格、公式位置、数值精度完全一致。
- 说明：`社零额（定）`与`汇总表（定）`的部分关键参数为**手工输入**，不依赖外部接口。
